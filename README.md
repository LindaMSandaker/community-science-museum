## About The Project

Semester project year 1, Front-end Development, Noroff VOC Oslo.

Make a website for the new Science Museum in town.

Language used; HTML and CSS, only Javascript is for hamburger menu.

## Getting Started

### Installation

2. Clone the repo
   ```sh

   ```
3. Install NPM packages
   ```sh
   npm install
   ```
